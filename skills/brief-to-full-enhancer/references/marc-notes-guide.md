# Brief-to-Full Enhancer: MARC Notes Field Guide

Reference for the 520 (Summary) and 505 (Contents Note) fields generated by
`enhance_record.py`.

---

## MARC 520 — Summary, Abstract, Scope and Content

### Indicator Values

| ind1 | ind2 | Meaning | Display label |
|------|------|---------|---------------|
| ` ` | ` ` | Summary (general) | "Summary:" |
| `1` | ` ` | Review | "Review:" |
| `2` | ` ` | Scope and content | "Scope and content:" |
| `3` | ` ` | Abstract | "Abstract:" |
| `4` | ` ` | Content advice | "Content advice:" |
| `8` | ` ` | No display constant | *(none)* |

`enhance_record.py` uses `ind1=" "` (general summary).

### Subfields

| Code | Meaning |
|------|---------|
| `$a` | Summary text (repeatable) |
| `$b` | Expansion (rarely used) |
| `$9` | `AI_QUICKCAT` — provenance stamp added by QuickCat |

### LC Practice

- 2–4 sentences for a monograph summary.
- Write in third person, past tense for historical works; present tense for reference.
- Do not begin with "This book…"; start with the subject.
- Avoid spoilers in fiction summaries.

### Example

```
520    $a A wealthy and mysterious man throws lavish parties at his Long Island
estate while pursuing a lost love across the class divide of 1920s America.
$9 AI_QUICKCAT
```

## MARC 505 — Formatted Contents Note

### Indicator Values

| ind1 | ind2 | Meaning |
|------|------|---------|
| `0` | ` ` | Complete contents, basic format |
| `0` | `0` | Complete contents, enhanced format (subfield-coded) |
| `1` | ` ` | Incomplete contents |
| `2` | ` ` | Partial contents |
| `8` | ` ` | No display constant |

`enhance_record.py` uses `ind1="0"` (complete contents).

### Subfields (basic format, ind2 = ` `)

| Code | Meaning |
|------|---------|
| `$a` | Formatted contents note text |
| `$9` | `AI_QUICKCAT` — provenance stamp |

### Formatting Conventions

- Chapters separated by ` -- ` (space, double hyphen, space).
- Each chapter title sentence-cased.
- Do not add chapter numbers if not present in the source.

### Example

```
505 00 $a Chapter 1. In my younger and more vulnerable years --
Chapter 2. Some time toward the end of August -- Chapter 3. There was music.
$9 AI_QUICKCAT
```

## When to Generate Each Field

| Condition | 520 | 505 |
|-----------|-----|-----|
| Record already has the field | Skip (use `--force` to overwrite) | Skip (use `--force`) |
| Material type is book | Yes | Yes if chapter structure found |
| Material type is journal | Scope note possible | No |
| Material type is ebook | Yes | Yes if structured |
